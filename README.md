# CarePlus Healthcare

A responsive healthcare website developed as part of the **MANTRA 2026 Summer School – Frontend Website Assignment**.

## 📌 Project Overview

CarePlus Healthcare is a static frontend website designed using HTML5, CSS3, and JavaScript. The website provides information about a healthcare center, its services, facilities, gallery, and an online appointment booking form.

## ✨ Features

* Responsive design for mobile, tablet, and desktop
* Home page with introduction and emergency contact
* About page with mission, vision, and facilities
* Medical services page
* Image gallery
* Appointment booking form with JavaScript validation
* Basic SEO implementation
* Clean navigation menu and footer

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub
* Netlify
